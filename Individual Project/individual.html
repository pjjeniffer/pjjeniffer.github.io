What is HTML and CSS?

HTML is HyperText Markup Language,The hypertext markup language is an application under the standard generic markup language, which is also a specification, a standard that marks the various parts of a web page to be displayed by marking symbols.CSS is Cascading Style Sheets, A computer language used to display file styles such as HTML or XML.
CSS can be used to not only modify pages in a static way, but also to format the elements of the page dynamically with various scripting languages.
