What is github?

GitHub is a hosting platform for open source and proprietary software projects.
In addition to hosting git code warehouse and basic Web management interface, also provides subscription, discussion groups, text rendering, online file editor, collaborative graph (report), and share code snippet (Gist), and other functions.
