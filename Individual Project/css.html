CSS：Three kinds of style

1.External style sheet

Write a .css file,then use link tag between the head tag to use the External style sheet.
it is so convenient for making several same theme e=websites

2.Internal style sheet

Use style tag which type="text/css" to use Internal style sheet.

3.Inline style

Use style (style) properties within associated labels.
